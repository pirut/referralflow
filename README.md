## The Problem

Local businesses survive on referrals, but:
- Staff forget to ask "who referred you?"
- Spreadsheets get messy
- Rewards are inconsistent
- No insight into who's bringing business

## The Solution

**ReferralFlow** gives businesses:
- Unique referral codes for each customer
- Simple tracking dashboard
- Automated reward configuration
- Analytics on top referrers
